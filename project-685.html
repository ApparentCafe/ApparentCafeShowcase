Project Euler Math Problems

Project Euler is a computational math inspired hobby project started and maintained by Colin Hughes. It contains hundreds of math problems that require 2 things, an understanding of an interesting math concept, and the ability to apply that concept to efficiently calculate the answer to the problem. My interest in this website comes from an enjoyment in finding creative and efficient algorithms that implement math concepts I've learned about or researched.
Some problems build off one another, but I have completed over 100 of them so far and it has been an informative and useful pass-time.
